# ThreatOS Wallpapers

This directory contains wallpaper images for ThreatOS, organized by category and resolution.

## Directory Structure

```
wallpapers/
├── default/       # Default wallpapers
├── abstract/      # Abstract designs
├── minimal/       # Minimalist wallpapers
├── nature/        # Nature-inspired wallpapers
└── dark/          # Dark-themed wallpapers
```

## Wallpaper Specifications

### Resolutions

Wallpapers should be provided in the following standard resolutions:

- **HD**: 1920×1080 (16:9)
- **FHD**: 2560×1440 (16:9)
- **QHD**: 3440×1440 (21:9)
- **4K UHD**: 3840×2160 (16:9)
- **5K**: 5120×2880 (16:9)
- **8K UHD**: 7680×4320 (16:9)

### File Naming Convention

Wallpapers should follow this naming convention:

```
[category]-[name]-[resolution].[format]
```

- `category`: The category of the wallpaper (e.g., `default`, `abstract`, `minimal`)
- `name`: A descriptive name for the wallpaper (e.g., `mountain`, `circuits`, `gradient`)
- `resolution`: The resolution of the wallpaper (e.g., `1920x1080`, `3840x2160`)
- `format`: The file format (e.g., `jpg`, `png`)

### Examples

- `default-threatos-3840x2160.jpg` - Default ThreatOS wallpaper in 4K
- `abstract-circuits-2560x1440.jpg` - Abstract circuit design in 1440p
- `minimal-gradient-5120x2880.jpg` - Minimal gradient wallpaper in 5K

## Design Guidelines

1. **Aspect Ratio**: Maintain the correct aspect ratio for each resolution.
2. **Color Scheme**: Use the official ThreatOS color palette (see `../colors/color-palette.gpl`).
3. **Quality**: Use high-quality source images with minimal compression artifacts.
4. **File Size**: Optimize file sizes without sacrificing quality (JPEG quality 85-90%).
5. **Branding**: Include subtle ThreatOS branding when appropriate, but avoid being intrusive.

## Contributing

1. Create a new branch for your wallpaper submission.
2. Place your wallpaper in the appropriate category directory.
3. Follow the file naming convention.
4. Include a brief description of the wallpaper in your pull request.
5. Ensure you have the rights to share the wallpaper under the project's license.
