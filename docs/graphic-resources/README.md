## Color Schemes

ThreatOS uses a carefully selected color palette to ensure consistency across the user interface. The color schemes are available in multiple formats:

- `colors/color-palette.gpl` - GIMP color palette
- `colors/color-palette.svg` - Vector format of the color palette
- `colors/terminal-color-scheme.json` - Terminal color scheme

### Primary Colors

- **Primary Blue**: `#367BF0` - Used for primary actions and highlights
- **Success Green**: `#0CA15E` - Used for success states and confirmations
- **Warning Orange**: `#FD7D00` - Used for warnings and attention-required states
- **Error Red**: `#D41919` - Used for errors and destructive actions

## Icons

ThreatOS includes a comprehensive set of icons for applications, system functions, and file types. The icon set follows a consistent design language that aligns with the overall aesthetic of the operating system.

### Icon Sizes

Icons are provided in the following standard sizes:

- 16x16px
- 24x24px
- 32x32px
- 48x48px
- 64x64px
- 128x128px
- 256x256px
- 512x512px (for application launchers)

### Icon Themes

- **Default** - The standard icon theme for ThreatOS
- **Dark** - A dark variant of the default theme
- **High Contrast** - High contrast version for accessibility

## Wallpapers

ThreatOS includes a collection of high-quality wallpapers that can be used as desktop backgrounds. These are available in various resolutions to support different display sizes.

### Categories

- **Default** - The default ThreatOS wallpaper
- **Abstract** - Abstract designs featuring the ThreatOS color scheme
- **Minimal** - Clean and minimal wallpapers
- **Nature** - Nature-inspired wallpapers
- **Dark** - Dark-themed wallpapers

## Themes

### GTK Theme

The GTK theme controls the appearance of all GTK-based applications in ThreatOS. It includes:

- Light and dark variants
- Support for GTK 3 and GTK 4
- Custom widgets and controls

### Icon Theme

- Consistent icon set for all system and application icons
- Multiple sizes for different use cases
- Support for symbolic icons

### Cursor Theme

- Custom cursor set matching the ThreatOS design language
- Multiple cursor sizes for different display resolutions
- High-DPI support

## Usage Guidelines

### Color Usage

- Use primary blue for primary actions and important elements
- Use success green for positive actions and confirmations
- Use warning orange for warnings and attention-required states
- Use error red for errors and destructive actions

### Icon Usage

- Use appropriate icon sizes for different contexts
- Maintain consistent spacing around icons
- Use the correct icon variant (filled/outlined) based on the context

### Typography

- Use appropriate font weights for different hierarchy levels
- Maintain proper line heights and letter spacing for readability
- Use proper contrast ratios for text on colored backgrounds

## Contributing

We welcome contributions to the ThreatOS graphic resources. Please follow these guidelines:

1. Follow the existing design language and style
2. Maintain consistency with the color scheme
3. Provide assets in both vector (SVG) and raster (PNG) formats when applicable
4. Include all necessary sizes and variants
5. Update the documentation accordingly
