# ThreatOS Branding

This directory contains branding materials for ThreatOS, including logos, banners, and social media assets.

## Directory Structure

```
branding/
├── logos/           # Logo variations
│   ├── color/      # Full-color logos
│   ├── white/      # White logos (for dark backgrounds)
│   ├── black/      # Black logos (for light backgrounds)
│   └── monochrome/ # Single-color logos
├── banners/        # Web and event banners
│   ├── web/        # Website banners
│   ├── social/     # Social media banners
│   └── events/     # Event banners
└── social-media/   # Social media assets
    ├── avatars/    # Profile pictures
    └── covers/     # Cover images
```

## Logo Usage

### Primary Logo

The primary ThreatOS logo should be used in most cases. It consists of the ThreatOS wordmark and symbol.

### Clear Space

Maintain clear space around the logo equal to the height of the "T" in the wordmark.

### Minimum Size

- **Print**: 1.5 inches wide
- **Digital**: 150 pixels wide

### Color Variations

- **Color**: Use on light backgrounds
- **White**: Use on dark or colored backgrounds
- **Black**: Use on light backgrounds when color is not available
- **Monochrome**: For single-color applications

### Incorrect Usage

- Do not rotate the logo
- Do not apply effects like drop shadows or bevels
- Do not stretch or distort the logo
- Do not change the colors of the logo
- Do not place the logo on busy backgrounds without sufficient contrast

## Typography

### Primary Font: Inter

- **Weights**: Light (300), Regular (400), Medium (500), SemiBold (600), Bold (700)
- **Usage**: Headings, body text, UI elements

### Monospace Font: Fira Code

- **Weights**: Regular (400), Medium (500), SemiBold (600)
- **Usage**: Code, terminal, technical documentation

## Brand Voice

### Tone

- **Professional**: Expert and authoritative
- **Approachable**: Friendly and helpful
- **Direct**: Clear and concise
- **Empowering**: Enabling and supportive

### Messaging

- Focus on security and privacy
- Emphasize open-source values
- Highlight community and collaboration
- Communicate technical capabilities clearly

## Social Media

### Profile Pictures

- Use the square logo with padding
- Minimum size: 400x400 pixels
- Recommended size: 1000x1000 pixels

### Cover Images

- Dimensions vary by platform
- Include the ThreatOS logo and tagline
- Use the brand colors and imagery

## Print Materials

### Business Cards

- Size: 3.5" x 2" (standard US)
- Bleed: 0.125" on all sides
- Safe zone: 0.25" from edges

### Letterhead

- Size: 8.5" x 11" (US Letter)
- Margins: 1" on all sides
- Include logo and contact information

## Digital Assets

### Favicon

- Size: 32x32 pixels
- Format: .ico or .png
- Use the ThreatOS symbol

### Email Signature

- Include name, title, and contact information
- Use the official email template
- Keep it simple and professional
